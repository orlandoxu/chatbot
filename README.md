# chatbot

> This is a wechat bot offer virtual coins price.

### install  
You need nodejs >= 14.0.0
```
git clone https://github.com/orlandoxu/chatbot.git
cd chatbot
npm install
cp config.yaml.example config.yaml
node app.js
```
